# Tech Stack

This project was developed using a Python-based data science environment focused on environmental data analysis and machine learning modeling.

---

## Programming Language

- **Python 3.x**

---

## Data Processing & Manipulation

- **pandas** – data cleaning, merging, time-series handling
- **NumPy** – numerical operations and array computations

---

## Data Visualization

- **matplotlib** – static visualizations and seasonal trend plots
- **seaborn** – correlation matrices and statistical visualization

---

## Machine Learning

- **scikit-learn**

  - Model implementation (SVR, MLP, Random Forest)
  - Train/test split
  - K-Fold cross-validation
  - Performance metrics (RMSE, MAE, R²)

- **XGBoost** – gradient boosting implementation
- **LightGBM** – efficient gradient boosting framework

---

## Time-Series & Feature Engineering

- Lag feature construction
- Rolling window statistics
- Seasonal binary encoding
- Interaction feature generation

---

## Model Evaluation & Validation

- Cross-validation (K-Fold)
- Out-of-sample testing
- Model comparison framework
- Feature importance analysis (tree-based models)

---

## Data Sources & External Tools

- Polish State Environmental Monitoring (GIOŚ)
- Meteostat API (meteorological data retrieval)

---

## Workflow & Methodology

- End-to-end ML pipeline:

  1. Data acquisition
  2. Cleaning & synchronization
  3. Feature engineering
  4. Model training
  5. Evaluation & comparison
  6. Interpretation

- Emphasis on reproducibility and structured experimentation
